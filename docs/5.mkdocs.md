## 主题

#### 预览图

![image-20180721171904260](imgs/image-20180721171904260.png)

1.安装主题

```
pip install mkdocs-material
```

2.mkdocs.yml文件修改主题

```
theme: material
```



## CentOS环境

#### 运行项目

```
mkdocs serve --dev-addr=10.211.55.3:8230 #浏览器访问：10.211.55.3:8230
```

#### 生成项目

```
mkdocs build
```
